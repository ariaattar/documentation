## Overview

Browse your connected databases, select tables, and view raw data for all your connected data sources.

## Features

- Select databases and tables from any connected data source
- View table schemas and column definitions  
- Browse raw data with pagination
- Explore relationships between tables

<Note>
**Data Privacy**: Only metadata is accessed by default unless you explicitly enable data sampling.
</Note>
